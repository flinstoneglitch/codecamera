Code Camera Privacy Policy

Last updated: June 2026

Code Camera processes images locally on your device to extract text using Google ML Kit OCR technology.

Information Collection

Code Camera does not collect, store, transmit, or sell personal information.

Photos and Camera Access

Photos selected by the user and images captured with the camera are processed locally on the device for text recognition purposes only.

Data Storage

No images or extracted text are uploaded to any external server by Code Camera.

Third-Party Services

Code Camera uses Google ML Kit for on-device text recognition.
